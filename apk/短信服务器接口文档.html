短信服务器接口文档



使用说明

请求地址：http://host:port/serviceName

本服务为局域网络网络服务，请求方式为http的GET请求暂不支持POST及其他类型的请求

服务的IP地址请在APP启动后，图形界面中获取

请求中的host和port均可在APP启动后获取得到，service请参考常用的服务和名称列表

请在第一次安装APP时测试短信权限，避免无法发送

请在第一次使用时设置本机号码、接收服务的IP和端口信息



支持的服务预览

服务名称 服务描述 请求方式 备注
serviceSendMsg 发送实时短消息 GET 无
ServiceSendDelayMsg 发送延时短消息 GET 无
serviceQuarySendMsgHistory 查询最近的短消息记录，默认存储最近三个月内的信息 GET 无
serviceQuaryTransponDrecord 查询最近的短消息转发记录，记录默认保存6个月 GET 无



1. 短信实时发送接口

服务名称：serviceSendMsg

完整服务实例：

http://192.168.1.103:8090/serviceSendMsg?phoneNumber=13212345689,13612345678&msgContent=液位报警&msgType=000001&msgName=液位报警

上面的短信息完成完成了一个及时短信息的发送，其中192.168.1.103:8090 是在APP启动后，屏幕中获取到的；phoneNumber后面传输手机号，多个手机号请用逗号分隔；msgContent是短信的信息体，支持多文字（注：不同运营商对文字的长度规定不同，短信的信息体过长时，文本将会被分成多条发送，例如要150字短信，以移动运营商为例，将会分为两条70字和一条10字的短信），为节省开支，请尽量将短信控制在70字符内，标点将视为一个字符。

服务请求参数：

参数名称 作用 是否必传 参数长度限制 默认参数 备注
phoneNumber 需要发送的目标对象的手机号（目前仅支持11大陆移动、联通、电信手机用户） 是 不限长度 无 多手机号请使用逗号分隔
msgContent 短信的信息体 是 不限长度 无 长度过长的短信将会被运营商拆分成70字符每条的若干条短信
msgType 信息类别代码 否 15字符内 null
msgName 信息类别名称 否 15字符内 其他

服务返回参数：

参数名称 作用 参数类型 参数取值及其含义 备注
code 请求结果返回码 int 200：成功;500：失败；800：请求服务器不支持的请求方式；900：服务不支持或不存在
msg 请求的结果信息说明 String 便于排查错误  
time 服务器处理当前请求的时间 String 返回一个 2019年01月12日 12:12:12 格式的时间串  
phoneNumber 需要发送的目标对象的手机号 String 13212345687  
msgContent 短信的信息体 String 短信的信息体  
msgType 信息类别代码 String 类别代码  
msgName 信息类别名称 String 类别名称   
{
    "code": 200,
    "content": "这里可以发送报警信息哦",
    "msg": "短信发送成功",
    "phoneNumber": "13621114221",
    "time": "2019年07月14日 18:30:48"
}



2. 延时发送短信发送接口

服务名称：ServiceSendDelayMsg

完整服务实例：

http://192.168.1.103:8090/ServiceSendDelayMsg?phoneNumber=13212345689,13612345678&msgContent=液位报警&msgDelayTimems=1000&msgType=000001&msgName=液位报警

服务请求参数：

参数名称 作用 是否必传 参数长度限制 默认参数 备注
phoneNumber 需要发送的目标对象的手机号（目前仅支持11大陆移动、联通、电信手机用户） 是 不限长度 无 多手机号请使用逗号分隔
msgContent 短信的信息体 是 不限长度 无 长度过长的短信将会被运营商拆分成70字符每条的若干条短信
msgDelayTimems 延时发送短消息，单位毫秒 否 不超过long取值范围 默认为0，立即发送 为节约资源，系统每6s定时循环查找需要发送的讯息，所以可能存在最大误差为6s；msgDelayTimems的值为0或者补传默认发送及时短讯息
msgType 信息类别代码 否 15字符内 null
msgName 信息类别名称 否 15字符内 其他

服务返回参数：

参数名称 作用 参数类型 参数取值及其含义 备注
code 请求结果返回码 int 200：成功;500：失败；800：请求服务器不支持的请求方式；900：服务不支持或不存在
msg 请求的结果信息说明 String 便于排查错误  
time 服务器处理当前请求的时间 String 返回一个 2019年01月12日 12:12:12 格式的时间串  
phoneNumber 需要发送的目标对象的手机号 String 13212345687  
msgContent 短信的信息体 String 短信的信息体  
msgType 信息类别代码 String 类别代码  
msgName 信息类别名称 String 类别名称
